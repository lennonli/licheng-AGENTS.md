
你现在协助我起草中国境内IPO项目法律尽职调查报告。我将提供尽调报告模板、上一版本报告、企业信用报告及尽调底稿。请直接修改Word文件并遵守以下要求：

一、基本原则

1. 以我最后修改或指定的版本为唯一底稿，保留我的人工修改。
2. 参照模板的结构、详细程度和列表方式，根据目标公司资料重新起草，不机械替换名称。
3. 仅修改我指定的章节，其他部分不得改动。
4. 事实必须有资料依据。无法确认的名称、日期、金额、比例、任职期间、亲属关系、控制关系等，使用明确的【待公司确认】或其他【】占位，不得推测。
5. 正文统一表述为“根据公司提供的资料及公开信息核查”，不要写明企查查等商业数据库名称。
6. 不得因为我提供了某家企业的报告，就推定该企业与公司、实际控制人、股东或董监高存在关联关系；必须核对股权、任职及控制路径。

二、释义

根据目标公司的企业信用报告和现有资料，删除模板旧项目释义，重新整理：

1. 公司全称及简称；
2. 控股股东、实际控制人、持股平台和持股5%以上股东；
3. 现存附属公司、境外主体及分公司；
4. 集团主体的范围；
5. 本项目、本次发行上市、本报告；
6. 报告期、报告出具日及其他通用释义。

公司、股东、附属公司和分公司的名称必须使用工商登记全称。集团主体范围应与报告第四部分及附件保持一致。无法确认的主体关系使用【待公司确认】。

三、“一、公司的基本情况”

参照模板结构，根据公司营业执照、章程、企业信用报告及工商资料重新起草，至少包括：

1. 公司的基本登记情况，以表格列明企业名称、统一社会信用代码、法定代表人、注册资本、实缴资本、企业类型、住所、成立日期、营业期限、经营范围和登记状态；
2. 公司的设立情况，包括设立时间、设立方式、设立时股东、认缴及实缴出资情况；
3. 公司设立后的历次名称、注册资本、股东、股权及企业类型变更，按照时间顺序列表；
4. 公司目前的股权结构，并与第三部分股东情况保持一致；
5. 分公司、迁址、更名或其他重要工商登记事项；
6. 公司有效存续情况及初步法律结论。

历史沿革只能根据工商档案、章程、股东会决议、出资文件等资料起草。企业信用报告未披露完整变更内容的，应使用【待公司提供工商档案核验】，不得自行补足。

四、其他章节

1. 股东和实际控制人：逐一列表说明每名股东的基本情况；法人及合伙企业列明登记信息和股东或合伙人结构；单独分析控股股东、共同实际控制人、一致行动关系、员工持股平台及股权质押冻结情况。
2. 附属公司：按照报告顺序逐家列表，核对持股比例、历史沿革、经营状态及注销情况。
3. 关联方：区分关联自然人、5%以上股东、持股平台、附属公司、实际控制人及董监高关联企业和历史关联方，并写明认定依据。
4. 知识产权：商标、专利和软件著作权列入附件；商标保留图案；专利仅列已授权项目；共同权利人完整列示；重复项目去重；权利限制单独披露。
5. 公司治理、诉讼仲裁及行政处罚：按照“事实概况—法律分析—建议”起草，区分公司、附属公司及股东或实际控制人事项，不得混同。
