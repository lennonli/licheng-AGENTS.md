# 法律尽调报告提示词

本文件用于沉淀法律尽职调查报告的专项提示词。当前版本仅收录“股权历史沿革”部分；后续可按同一格式继续追加主体资格、业务资质、重大合同、知识产权、劳动用工、诉讼仲裁、行政处罚等章节。

## 一、使用方式

- 本提示词应与 `法律业务通用指令.md` 叠加使用；如具体项目已有客户身份、交易背景、报告模板或特别要求，应以具体项目指令优先。
- 使用前应将目标公司名称、工商内档 PDF 路径、参考报告 DOCX 路径、底稿输出目录、报告输出目录等占位内容替换为真实信息。
- 除非用户另有说明，本部分仅处理股权历史沿革相关事项，不扩展处理经营范围、注册地址、法定代表人、董事监事高级管理人员、备案章程文本等非股权事项。
- 对无法从底稿中核实的事实、金额、比例、日期、付款、纳税、出资实缴情况，应明确标注 `【待核验】`，不得自行补全或推断。

## 二、股权历史沿革部分

### （一）任务目标

请根据目标公司的工商内档资料、已有底稿及参考尽调报告的表述方式，完成以下工作：

1. 梳理目标公司自设立以来与股权变动相关的历史沿革。
2. 仅提取和分析股东、股权比例、认缴出资额、实缴出资、注册资本、股权转让、增资、减资、股改、代持还原、员工持股平台或其他影响股权结构的事项。
3. 不写经营范围变更、住所变更、法定代表人变更、董监高备案、一般章程备案等与股权结构无关的内容；如某项非股权信息仅为解释股权变动所必需，可在最小范围内作为背景说明。
4. 将每一次股权变动对应的工商底稿从原始 PDF 中拆分出来，整理为单独 PDF。
5. 参照参考尽调报告第二章或历史沿革章节的表达方式，起草目标公司尽调报告中的“历史沿革”部分；报告初稿只写历史沿革部分，其他章节不要新增、改写或扩展。
6. 根据核查中发现的缺失材料、证据链断点或待确认事项，另行形成具体详细的补充材料清单。

### （二）输入材料

请按下列信息开展工作：

- 目标公司名称：`〔填写目标公司名称〕`
- 工商内档 PDF：`〔填写工商内档 PDF 绝对路径〕`
- 参考尽调报告 DOCX：`〔填写参考报告 DOCX 绝对路径〕`
- 已有历史沿革梳理表或底稿目录：`〔如有则填写路径；如无则写“无”〕`
- 底稿 PDF 输出目录：`〔填写输出目录〕`
- 历史沿革初稿输出目录：`〔填写输出目录〕`

### （三）核查与梳理要求

1. 以工商内档为主要事实来源，重点核对设立登记、变更登记申请书、股东会决议、董事会决议、章程、章程修正案、股权转让协议、增资协议、验资报告、出资证明、股东名册、营业执照及企业信用信息公示资料。
2. 如 PDF 为扫描件，应先进行 OCR 识别，并人工复核每一次股权变动的关键事实，包括变更日期、决议日期、转让方、受让方、转让比例、转让出资额、注册资本、认缴出资额、实缴出资额、变更后股权结构。
3. 将公司设立作为历史沿革起点单独梳理；设立资料中应重点提取设立时间、注册资本、初始股东、认缴出资额、持股比例、出资期限、公司章程及设立登记文件。
4. 对历次股权变动按时间顺序编号，不得遗漏中间变更。工商变更日期、股东会决议日期、协议签署日期、营业执照核发日期不一致时，应分别说明并优先使用能准确反映交易或决议发生的日期作为正文叙述基础。
5. 每一次股权变动均应形成一条结构化记录，至少包括：
   - 序号；
   - 变更时间；
   - 变更事项；
   - 变更依据文件；
   - 变更前股权结构；
   - 变更后股权结构；
   - 对应工商内档页码；
   - 对应拆分底稿 PDF 文件名；
   - 待核验事项。
6. 如发现股权转让价款、税款缴纳、出资实缴、优先购买权放弃、配偶同意、国资审批、外资审批、员工持股平台内部份额、代持安排等材料缺失，应放入补充材料清单，不得在正文中作无依据的确定结论。

### （四）底稿 PDF 整理要求

1. 将历史沿革相关底稿从原工商内档 PDF 中单独拆分，不纳入经营范围、地址、法定代表人、董监高等无关变更材料。
2. 每一次股权变动制作一个独立 PDF；公司设立亦作为一个独立 PDF。
3. 每个 PDF 应尽量包含该次股权变动的完整证据链，包括但不限于变更登记申请书、股东会决议、股权转让协议、章程或章程修正案、验资报告或出资证明、股东名册、营业执照、准予变更登记通知书等。
4. PDF 文件命名规则为：`序号-日期-变更事项.pdf`。
5. 日期格式优先使用 `YYYYMM`；如需要精确到日，可使用 `YYYYMMDD`。
6. 命名示例：`1-201703-第一次股权变动（A转让10%股权给B）.pdf`。
7. 文件名应简洁、可检索、能看出本次变更实质；涉及多个转让方或受让方时，可概括为 `A等转让合计30%股权给B等`。
8. 拆分后应复核 PDF 页数、页面方向、可读性、是否漏页、是否误纳入无关材料，并制作或更新“股权历史沿革梳理表”。

### （五）报告正文写作要求

1. 参照参考尽调报告中历史沿革章节的层级、语气和表达方式，但不得复制参考报告中的事实内容。
2. 建议章节结构如下：
   - `二、公司的历史沿革`
   - `（一）公司设立及历次股权变动`
   - `1、公司设立`
   - `2、第一次股权变动`
   - `3、第二次股权变动`
   - `（二）历史沿革相关核查事项`
3. 正文开头可采用类似表述：`根据公司提供的工商内档资料、公司章程、股东会决议、股权转让协议及本所律师核查，公司自设立以来的股权历史沿革情况如下：`
4. 每一次股权变动应按“事实依据 + 变更内容 + 变更后股权结构”的顺序撰写：
   - 事实依据：说明股东会决议、协议、章程修正案、验资报告、工商登记或营业执照等文件；
   - 变更内容：说明转让方、受让方、转让比例、对应出资额、增资或减资金额、注册资本变化；
   - 股权结构：用表格列示变更后的股东、认缴出资额及持股比例。
5. 股权结构表建议字段如下：
   - 序号；
   - 股东姓名或名称；
   - 认缴出资额（万元）；
   - 实缴出资额（万元，如可核实）；
   - 持股比例；
   - 出资方式；
   - 出资期限；
   - 备注。
6. 对 1,000 及以上数字使用千位分隔符；金额、比例、日期必须与底稿一致。
7. 如底稿未显示股权转让价款支付、个人所得税或印花税缴纳、出资实缴、优先购买权放弃等情况，应在“历史沿革相关核查事项”中列明，不得在正文直接写成“已依法履行完毕”。
8. 如存在代持、回购、股权激励、员工持股平台、实控人变化、增资估值异常、出资瑕疵、减资债权人通知、国资或外资审批等特殊情况，应单独设段说明，并提出需要补充核查的文件。
9. 报告文字应客观、克制、可追溯，避免使用“合法有效”“不存在争议”等最终结论性表述，除非底稿证据链充分且已完成必要外部核验。

### （六）历史沿革补充材料清单

如现有工商内档或底稿不足以支撑完整结论，应要求公司补充提供以下材料。实际清单应结合已发现问题逐项细化，不要只写概括性表述。

1. 工商登记及章程资料
   - 公司设立及历次股权变动的完整工商内档；
   - 历次工商变更登记申请书、准予设立或变更登记通知书、营业执照；
   - 历次公司章程、章程修正案、股东名册、出资情况表；
   - 最新公司章程、最新营业执照、最新企业信用信息公示系统报告；
   - 如工商内档缺页、扫描不清或页码不连续，补充清晰完整扫描件。
2. 出资实缴及注册资本资料
   - 历次出资的银行进账单、回单、验资报告、验资附表、询证函；
   - 会计凭证、记账凭证、财务账簿、审计报告或资本公积入账凭证；
   - 非货币出资的评估报告、权属证明、交割或过户文件、发票及入账凭证；
   - 出资期限、逾期出资、抽逃出资或瑕疵出资的说明及整改证明；
   - 增资、减资相关的股东会决议、债权人通知、公告、清偿或担保说明。
3. 股权转让交易资料
   - 历次股权转让协议及补充协议；
   - 转让价款支付凭证、银行流水、收款确认书、债权债务抵销协议；
   - 股权转让定价依据、评估报告、审计报告、投融资协议或估值说明；
   - 未支付、低价转让、零对价转让、亲属或关联方转让的原因说明及相关证明；
   - 股权交割、工商变更完成及股东名册更新的确认文件。
4. 税费缴纳资料
   - 自然人股东股权转让个人所得税申报表及完税证明；
   - 企业股东股权转让所得税处理资料；
   - 印花税申报及缴纳凭证；
   - 税务机关出具的涉税事项证明或完税情况说明；
   - 如未缴税或无法提供完税凭证，说明原因、主管税务机关沟通情况及后续补缴安排。
5. 股东内部决策及权利放弃资料
   - 转让方、受让方内部决策文件，包括股东会、董事会、合伙人会议或投资决策委员会决议；
   - 其他股东放弃优先购买权的书面确认；
   - 自然人股东配偶同意函或婚姻财产归属说明；
   - 国资、外资、上市公司、基金、事业单位或其他特殊主体股东所需审批、备案或授权文件；
   - 股东签署文件的授权委托书、身份证明、主体资格证明及签字盖章页。
6. 股权权属及限制资料
   - 各现有及历史股东关于股权权属清晰、不存在代持、信托持股、委托持股、收益权安排的确认函；
   - 不存在股权质押、冻结、查封、权属争议、回购安排、对赌安排或其他权利限制的确认函；
   - 股权出质登记查询结果、司法冻结查询结果、诉讼仲裁及执行信息查询结果；
   - 如曾存在代持或股权权属纠纷，补充代持协议、解除协议、还原文件、付款凭证、税务处理资料及纠纷解决文件。
7. 员工持股平台及激励安排资料
   - 员工持股平台合伙协议、合伙人名册、出资证明、份额转让文件；
   - 股权激励计划、授予协议、退出协议、回购协议、归属或解锁安排；
   - 激励对象出资或受让价款支付凭证、税务处理资料；
   - 平台普通合伙人、执行事务合伙人及实际控制关系说明；
   - 离职员工份额退出、代持或预留份额的处理文件。
8. 实际控制人及特殊历史事项资料
   - 公司实际控制人认定说明及历次变化情况；
   - 一致行动协议、表决权委托协议、投票权安排、控制权稳定性承诺；
   - 红筹拆除、境外架构还原、VIE 解除、股改、股份制改造、历史沿革瑕疵整改资料；
   - 历史股东失联、注销、死亡、继承、合并分立等特殊事项的证明文件；
   - 公司及主要股东就历史沿革真实性、完整性、无重大争议出具的书面说明。

### （七）输出成果与验收要求

最终应交付以下成果：

1. 股权历史沿革梳理表，至少包括序号、日期、变更事项、变更依据、变更后股权结构、原 PDF 页码、对应底稿 PDF、待核验事项。
2. 历次股权变动底稿 PDF，按 `序号-日期-变更事项.pdf` 命名，每一次股权变动一个 PDF。
3. 法律尽调报告历史沿革部分初稿，按项目命名规范生成 Word 文件；该初稿只写历史沿革部分，不扩展其他章节。
4. 历史沿革补充材料清单，逐项列明需补充材料、对应核查目的、涉及的具体股权变动或历史事项。
5. 交付前应复核：
   - 股权变动是否按时间顺序完整覆盖；
   - 每个事实是否能对应到底稿页码或拆分 PDF；
   - 拆分 PDF 是否可打开、可读、无漏页、无明显误纳入；
   - 报告正文是否排除了非股权事项；
   - `【待核验】` 事项是否均已进入补充材料清单。

## 三、后续章节扩展格式

后续新增其他尽调章节时，建议沿用以下结构：

1. 任务目标；
2. 输入材料；
3. 核查与梳理要求；
4. 底稿整理要求；
5. 报告正文写作要求；
6. 补充材料清单；
7. 输出成果与验收要求。
